Methods

``BaseGeometry`` has two methods, ``integer_validator`` and ``area`` methods

integer_validator
----------------
The validator function should take a string as the first argument,
and an integer as the second argument. The first argument is assumed to always be a string:
If the parameters are valid, the function does nothing:

    >>> bg.integer_validator('name', 2)

The second argument must be an integer:

    >>> bg.integer_validator(2, 'name')
    Traceback (most recent call last):
        ...
    TypeError: 2 must be an integer


    >>> bg.integer_validator('name', 'name')
    Traceback (most recent call last):
        ...
    TypeError: name must be an integer


    >>> bg.integer_validator('name', [1,2,3])
    Traceback (most recent call last):
        ...
    TypeError: name must be an integer


Only the ``name`` and ``value`` should be provided. Nothing more, nothing less:

    >>> bg.integer_validator('name', 2, 3)
    Traceback (most recent call last):
        ...
